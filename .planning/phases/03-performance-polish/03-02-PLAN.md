<name>Task 2.1: Verify CSS is optimized (10 minutes)</name>

<files>css/styles.css</files>

<action>
Verify and ensure CSS is optimized for performance:

1. Check CSS file size - should be under 50KB
2. Ensure fonts use `display=swap` in Google Fonts URL
3. Verify no duplicate or unused rules

**Specific Actions:**

1. Check CSS file size:
```bash
wc -c css/styles.css
```
Expected: Under 50000 bytes (50KB)

2. Check Google Fonts has display=swap:
```bash
grep "display=swap" css/styles.css
```
Expected: Found (prevents FOIT - flash of invisible text)

3. Check CSS is minified or reasonably compact (no excessive whitespace)
</action>

<verify>
<automated>wc -c css/styles.css | awk '{print $1}' && grep -q "display=swap" css/styles.css && echo "CSS_OK"</automated>
Expected: File size reported, then "CSS_OK" if display=swap found
</verify>

<done>
CSS file is under 50KB and uses display=swap for font loading optimization.
</done>
</task>

<name>Task 2.2: Verify HTML structure for performance (5 minutes)</name>

<files>index.html</files>

<action>
Verify HTML has proper performance optimizations:

1. Check viewport meta tag exists (mobile performance)
2. Check preconnect hints for Google Fonts
3. Check lazy loading on images below fold
4. Check script at end of body

**Specific Actions:**

1. Verify viewport meta tag:
```bash
grep 'meta name="viewport"' index.html
```

2. Verify preconnect for fonts:
```bash
grep 'rel="preconnect"' index.html
```

3. Verify lazy loading on non-hero images:
```bash
grep 'loading="lazy"' index.html
```
Expected: 3 instances (for the 3 non-featured artworks)

4. Verify script at end of body:
```bash
grep -A1 '</body>' index.html | grep 'script'
```
</action>

<verify>
<parameter name="automated">grep -q 'meta name="viewport"' index.html && grep -q 'loading="lazy"' index.html && echo "HTML_PERF_OK"</automated>
Expected: "HTML_PERF_OK" if both present
</verify>

<done>

</done>
