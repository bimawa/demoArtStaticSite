<task>
**Task 2.3: Add Subtle Hover Effects on Gallery Items (15 minutes)**

Implement GPU-accelerated hover effects that lift gallery items with a soft shadow, creating interactive feedback without distraction.

**Specific Actions:**
1. Add transition to `.artwork`:
   ```css
   transition: transform var(--transition-fast), box-shadow var(--transition-fast);
   ```

2. Add `.artwork:hover` rule:
   ```css
   .artwork:hover {
     transform: translateY(-4px);
     box-shadow: 0 8px 16px rgba(26, 26, 26, 0.12);
   }
   ```
   (Using soft black with 12% opacity for subtle shadow)

3. Add `.artwork:active` rule (for touch devices):
   ```css
   .artwork:active {
     transform: translateY(-4px);
     box-shadow: 0 8px 16px rgba(26, 26, 26, 0.12);
   }
   ```

4. Optional: Add `will-change: transform` to `.artwork` for GPU hint (use sparingly)

5. Test on mobile: Tap gallery items to see active state

**Success Criteria:**
- Gallery items lift on hover (desktop)
- Gallery items lift on tap (mobile)
- Shadow appears beneath lifted item
- Transition is smooth (0.2s ease)
- Effect is subtle and doesn't distract from artwork
- No layout shift when hovering
</task>
