# Artist Portfolio Website

## What This Is

A single-page portfolio website for a visual artist, functioning as a digital business card (визитка). The site showcases 4 artworks and provides contact information. Designed to be deployed on GitHub Pages.

## Core Value

A minimal, elegant digital presence that lets visitors quickly discover the artist's work and get in touch.

## Requirements

### Active

- [ ] Single-page layout with hero/about section
- [ ] Gallery section displaying 4 artworks
- [ ] Contact information section
- [ ] Mobile-responsive design
- [ ] Modern, clean visual style
- [ ] Works in all modern browsers

### Out of Scope

- [CMS or admin panel] — Static HTML only
- [Multiple pages] — Single page
- [E-commerce] — Portfolio only

## Context

- **Target deployment**: GitHub Pages
- **Tech constraint**: Must be pure HTML/CSS/JS (no build step) for GitHub Pages compatibility
- **Design**: Modern minimalist style

## Constraints

- **Deployment**: GitHub Pages — static files only
- **Browser support**: All modern browsers (Chrome, Firefox, Safari, Edge)
- **Mobile**: Fully responsive
